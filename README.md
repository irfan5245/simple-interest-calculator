[# Simple Interest Calculator

This project calculates simple interest based on principal, rate, and time.

## Formula
SI = (P × R × T) / 100

## Features
- Easy to use
- Beginner-friendly
- Command-line input

## How to Run
1. Clone the repository
2. Run the script:
   python simple_interest.py

## Example
Input:
Principal = 1000  
Rate = 5  
Time = 2  

Output:
Simple Interest = 100
